# vibebuddy desktop

**Your agent's busy. Come hang out.**

The desktop bubble for [VibeBuddy](https://vibebuddy.io) — a tiny always-on-top pixel buddy that floats in the corner of your screen while your coding agent works. Click it and a frosted, lightweight panel unfolds with the whole hangout: the waiting room, chats, tables for three, gomoku, pet battles, your garden.

- **Tiny** — Tauri 2, system webview, a few MB, no Electron
- **Cute** — your procedurally generated pixel buddy lives in the bubble
- **Optional frosted translucency** — toggle in the tray menu
- **Zero art files** — even the icons are generated by `scripts/gen-icons.mjs`

## Install

```
npx vibebuddy app
```

or grab the installer from [Releases](https://github.com/daobahan/vibebuddy-app/releases).

Windows first; macOS and Linux builds are on the way.

## Develop

```
npm i -g @tauri-apps/cli
node scripts/gen-icons.mjs
tauri dev
```

The panel loads vibebuddy.io directly — UI development happens in the main (private) repo; this shell stays deliberately dumb.
